Router

The router requires a server-side component because of the deeplinking. In order to get that to work, you should clone the repo and do the following:

  1. Clone the project.
  2. Install the Google App Engine SDK For Python.
  3. Run the SDK app to create symlinks.
  4. Add it to the GAE Launcher you downloaded (Existing app, (Ctrl|Cmd)+Shift+N, /path/to/router).
  5. Enjoy your local version of the router sample, probably at localhost:8080.
